# NetSentry

## CodeAlpha Basic Network Sniffer

> A Python-Based Network Traffic Analyzer built using Scapy

---

## 📖 Overview

NetSentry is a Python-based network traffic analyzer developed as part of the **CodeAlpha Cyber Security Internship**.

The application captures live network packets and analyzes them to help understand network communication, common protocols, and packet structures.

This project focuses on practical network monitoring and packet inspection using Python and Scapy.

---

## 🎯 Objectives

- Capture live network traffic
- Analyze network packets
- Identify common protocols
- Display meaningful packet information
- Improve understanding of TCP/IP networking

---

## ✨ Current Features

- Live packet capture
- Source IP detection
- Destination IP detection
- Basic protocol identification

---

## 🚀 Planned Features

## ⚙️ Installation

```bash
git clone https://github.com/Sheha-75/CodeAlpha_BasicNetworkSniffer.git

cd CodeAlpha_BasicNetworkSniffer

python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt
```

