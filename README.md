## 📌 About the Project

**NetSentry** is a Python-based network traffic analyzer developed as part of the **CodeAlpha Cyber Security Internship**.

This project captures and analyzes live network packets using **Scapy**, helping users understand network traffic, protocols, and packet structures through real-time monitoring.

The repository name follows the internship task requirements, while **NetSentry** serves as the project's professional identity.
